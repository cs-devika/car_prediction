# Car Prediction
## Project Overview
This case study is a ml based web application that predicts the price of a car based on user input.
<img src ="" width="500">
## Objective
- To build a machine learning model that predicts car price accurately and to analyze factors affecting car price.
- To apply data preprocessing and model evaluation techniques.
# Dataset
## Features used:
- Year
- Present_Price
- Kms_Driven
- Car_Age
# Technologies Used
- Python
- Pandas
- Numpy
- matplotlib
- Scikit-learn
- VS Code
- Flask
- HTML/CSS
# Project Workflow
- Understanding Data
- Data Cleaning
- Exploratory Data Analysis(EDA)
- Data Preprocessing
- Feature Engineering
- Model building
- Model Evaluation
- Prediction
- Model Deployment
# Machine Learning Models Used
- LinearRegression
# Model Evaluation
Evaluation metrics used:
- Mean Absolute Error
- Mean Squared Error
- R2 Score
# Results
- The model predicts car price with good accuracy
- LinearRegression give best performance
